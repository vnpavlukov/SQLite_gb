1. можно умножать матрицы на вектор
2. матрицу на матрицу, если число столбцов (м_1) = числу строк (м_2)

 1 2      1 0 0     1 0 4
 0 1  x   0 0 2  =  0 0 2
10 0               10 0 0

умножение матриц соответствует композиции двух линейных преобразований

3. складывать матрицы можно только одинакового размера

4. транспонирование - поворот относительно главной диагонали
                    - перемена стрко и слобцов местами


Определитель
 - можно задать только для квадратной матрицы
 - объём параллелограмма из n векторов
 - можно определить рекурсивно - теорема Гаусса


линейно-зависимые - лежат на одной прямлй
линейно-независимые - лежат не на одной прямой

определитель = 0 только тогда, когда строки матрицы линейно-зависимы


Ранг системы строк(столбцов) матрицы А - максимальное число ленейно независимых
строк(столбцов) среди них

Ранг по столбцам и по строкам всегда совпадают


определитель - объём параллелепипеда постронного по векторам,строкам матрицы А
Ранг - число линейно независимых строк или столбцов
     - характеризует кол-во информации в  матрице



Определение типа:
  Сравним ранк матрицы А и ранг (A|b) (расширенной матрицы А,
подставив правую часть матрицы b):
    if ранг расширенной матрицы равен рангу исходной, то есть хотя бы одно решение.

    Если ранг А равен числу неизвестных, то решение 1
    Если ранг А меньше числа неизвестных - то решений бесконечно много
